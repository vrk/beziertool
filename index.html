HTML5 <canvas> bezierCurve command generator

Intended For: People who already know JavaScript/Proccessing and want to draw curves without trying every possible Pixel Value.
Not Intended For: Auto-generating code for those who don't know what they're doing; replacing a full-fledged awesome drawing program like Illustrator.

This tool was originally created by Victoria Kirst in 2011 and I started working on it Dezember 2019.
